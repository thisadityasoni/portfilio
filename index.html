GroupcChat-analyzer

This project is a Streamlit-based application that provides comprehensive analysis of group chat data. It offers features like sentiment analysis, word clouds, emoji analysis, and activity maps. By processing uploaded chat files, the app generates detailed visualizations and insights, making it easier to understand messaging patterns and behaviors.

Library used : Pandas, Matplotlib, Seaborn, Streamlit.
AI chat application

This Flask-based project is a web application that integrates Firebase for authentication and database management. Users can sign up, log in, reset passwords, and access AI-powered chat features. The app provides a secure, session-based environment for users to interact with AI models while storing chat history in Firebase.

Library used : Firebase, flask, Google-genrativeai.
